# prompt-engineer

Generate comprehensive autonomous agent prompts for Claude Code from project ideas.

## Skills

### prompt-engineer

Turns a project description (from a one-liner to a detailed spec) into a fully-specified `.claude/prompts/init.md` file and startup prompts that make a Claude Code agent fully autonomous with quality gates, plugin infrastructure, stacked changes, code reviews, sub-agents, and iterative quality loops.

**Triggers:** "create a prompt for", "write an init prompt", "set up a new project", "bootstrap prompt", "generate a spec", "agentic prompt"

## What It Produces

1. **`.claude/prompts/init.md`** — A comprehensive specification and agent instruction file
2. **Startup prompts** — Copy-paste prompts for Session 1 (bootstrap) and Session 2+ (development)
