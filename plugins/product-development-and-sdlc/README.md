# product-development-and-sdlc

Iterative PRD and user story writing with structured SDLC workflows.

## Skills

### prd-writing

Guides writing product requirements documents and user stories through iterative refinement cycles rather than one-shot generation. Follows a seed-draft-research-review-refine loop to build specifications incrementally.

**Triggers:** "write a PRD", "create a spec", "write user stories", "define requirements", "iterate on a PRD"
