# OG Image Plugin

Generate Open Graph images using Claude Code.

## Overview

This plugin helps generate social media preview images (Open Graph images) for your content.

Source: https://www.reddit.com/r/ClaudeCode/comments/1pw2psj/this_is_how_i_generate_og_images_using_claude_code/

## Installation

See [Installation Guide](../../docs/installation.md) for all installation methods.

### Quick Install

```bash
# Via marketplace (recommended)
# Follow marketplace setup: ../../docs/manual-installation.md

# Or via GitHub
claude plugins install github:nsheaps/.ai/plugins/og-image

# Or locally for testing
cc --plugin-dir /path/to/plugins/og-image
```
