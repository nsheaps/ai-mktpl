A plugin that uses hooks + skills to manage a file for questions to the user as claude is running for use instead of AskUserQuestion so claude can keep running.

## Open questions from Claude to the user

Use this section like below to solicit any clarifications you need while working. The user will answer questions
in line as they appear in this file. The first question after the example is to show what it looks like after the
user has answered. Add any questions after the questions that already exist. When adding new questions, always be sure
there's an empty one at the end for you to add more questions to.

> === Question: What color is the sky
> Answer: ...
> ===

=== Question: What color is the sky
Answer: Blue
===

Answer: ...
===
