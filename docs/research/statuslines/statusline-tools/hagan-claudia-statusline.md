## Summary

High-performance Rust-based statusline for Claude Code with persistent stats tracking, progress bars, and optional cloud sync via Turso. Features SQLite-first persistence, 11 embedded themes, 5 layout presets, and burn rate calculation for tracking usage patterns.

## Key Features

- SQLite-first persistence
- Git integration
- Context progress bars
- Burn rate calculation
- 11 embedded themes
- 5 layout presets
- Optional Turso cloud sync
