## Summary

High-performance Go implementation of Claude Code utilities with statusline generation and MCP management. Features model and cost tracking, git awareness, environment context including Kubernetes and AWS, and token usage bars. No daemon required for operation.

## Key Features

- Model and cost tracking
- Git awareness
- Environment context (K8s, AWS)
- Token usage bars
- MCP management
- No daemon required

## Installation

go build && ./cc-tools statusline
