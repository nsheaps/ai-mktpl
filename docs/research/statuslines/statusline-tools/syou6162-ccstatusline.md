## Summary

Customizable statusline tool using YAML configuration. Process JSON input, execute shell commands, output formatted statusline with colors. Features template syntax for JSON data access, TTL-based caching, XDG-compliant paths, and ANSI color support.

## Key Features

- YAML configuration
- Template syntax for JSON data access
- Shell command execution
- TTL-based caching
- XDG compliant
- ANSI color support
