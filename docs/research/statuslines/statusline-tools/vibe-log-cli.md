## Summary

CLI tool for logging and analyzing Claude Code sessions with statusline integration. Provides intelligent session analysis and strategic guidance displayed directly in the statusline. Features parallel session analysis, standup summaries, ccusage integration, and automatic config backup on install.

## Key Features

- Local prompt analysis
- Actionable guidance in statusline
- Parallel session analysis
- Standup summaries
- ccusage integration
- Automatic config backup on install
