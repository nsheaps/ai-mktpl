## Summary

Lightweight statusline plugin for Claude Code CLI with multi-layer progress visualization and weekly usage tracking. Features a lightweight bash implementation, ccusage integration, privacy-first configuration approach, and support for both macOS and Linux.

## Key Features

- Multi-layer progress bars
- Weekly usage tracking
