## Summary

Simple statusline plugin using official usage API to show account-wide block usage and reset time in real-time. Features session context percentage display, cost tracking, and git branch info. Uses the official Claude API for accurate usage data.

## Key Features

- Official usage API integration
- Account-wide block usage
- Reset time display
- Session context percentage
- Cost tracking
- Git branch info
