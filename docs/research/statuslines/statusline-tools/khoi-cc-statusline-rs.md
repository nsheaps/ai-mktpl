## Summary

Rust version of Claude Code statusline with cost tracking. Based on steipete's popular gist. Features fast execution at approximately 85ms, git status display, and cost tracking. A lightweight Rust implementation for those who prefer compiled performance.

## Key Features

- Rust implementation
- Cost tracking
- Git status
- Fast execution (~85ms)

## Installation
