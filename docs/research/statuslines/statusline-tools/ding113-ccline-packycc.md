## Summary

Claude Code statusline tool written in Rust specifically for PackyCode with API quota monitoring. Features real-time quota monitoring, git integration, TUI configuration mode, and daily spending tracking for PackyCode API users.

## Key Features

- PackyCode API integration
- Real-time quota monitoring
- Git integration
- TUI configuration mode
- Daily spending tracking
