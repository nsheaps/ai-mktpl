## Summary

Customizable statusline with highly configurable git integration and directory display options. Features extensive git options including SHA display, working tree status, operation indicators, tag display, and stash count. Supports environment variable configuration and custom shell script extensions.

## Key Features

- Configurable git options (sha, working tree, operation, tag, stash count)
- Directory basename display
- Environment variable configuration
- Custom shell script extensions
