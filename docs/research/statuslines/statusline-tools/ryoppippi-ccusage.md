## Summary

CLI tool for analyzing Claude Code/Codex CLI usage from local JSONL files. Includes statusline integration (Beta) for compact usage display. Provides daily/monthly/session reports, 5-hour billing windows tracking, and model tracking with colorful table display.

## Key Features

- Daily/monthly/session reports
- 5-hour billing windows tracking
- Statusline integration for hooks
- Model tracking
- Date filtering
- Colorful table display

## Installation

```bash
bun x ccusage statusline
```
