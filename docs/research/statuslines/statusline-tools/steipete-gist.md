## Summary

Popular status bar script shared as a GitHub Gist. Averaging approximately 85ms execution time, this is the script that inspired the cc-statusline-rs Rust version. A lightweight bash implementation with git status (which can be disabled), session time tracking, and cost tracking.

## Key Features

- ~85ms execution time
- Git status (can be disabled)
- Session time tracking
- Cost tracking
