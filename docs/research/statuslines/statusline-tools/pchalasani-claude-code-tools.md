## Summary

Practical productivity tools for Claude Code including tmux-cli for terminal automation. While not primarily a statusline tool, it enables terminal control and can be used in conjunction with statusline displays. Features interactive script testing, debugger integration, and Claude-to-Claude communication.

## Key Features

- tmux-cli plugin
- Terminal automation
- Interactive script testing
- Debugger integration
- Claude-to-Claude communication

## Installation

```bash
claude plugin install tmux-cli@cctools-plugins
```
