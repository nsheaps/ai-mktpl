## Summary

Python-based statusline plugin displaying Claude API usage (model, cost, reset time) in real-time, powered by ccusage. Includes Windows batch file support for easy execution. Shows model name, cost information, and remaining time until usage reset.

## Key Features

- ccusage integration
- Model name display
- Cost information
- Remaining time display
- Windows batch file support

## Installation

```bash
python %USERPROFILE%\.claude\statusline-ccusage.py
```
