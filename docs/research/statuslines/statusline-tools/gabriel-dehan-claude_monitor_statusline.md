## Summary

A customizable Ruby status line for Claude Code that displays usage info with git status and model information. The only Ruby implementation in the ecosystem. Features multiple display modes, usage tracking with plan limits, and manual mock testing support for development.

## Key Features

- Git status with change indicators
- Model display
- Usage tracking with plan limits
- Multiple display modes
- Manual mock testing support

ruby ~/.claude/utils/claude_monitor_statusline/statusline.rb
