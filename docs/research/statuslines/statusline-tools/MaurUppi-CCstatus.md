## Summary

High-performance Claude Code statusline tool written in Rust with network detection, Git information, and real-time usage tracking. Provides detailed network statistics including P95 metrics, DNS, TCP, TLS, and TTFB measurements for monitoring Claude API connectivity.

## Key Features

- Network status indicator (P95 statistics)
- DNS, TCP, TLS, TTFB metrics
- Git integration
- JSONL log collection
- Error aggregation
- Cross-platform (Linux, macOS, Windows)
