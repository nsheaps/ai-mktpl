## Summary

High-performance Claude Code statusline tool written in Rust with Git integration, usage tracking, interactive TUI configuration, and Claude Code enhancement utilities. The Rust implementation provides excellent performance and cross-platform support.

## Key Features

- High performance Rust implementation
- Git integration with branch and status
- Interactive TUI configuration
- Real-time usage tracking
- Cross-platform support
