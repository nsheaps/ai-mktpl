Real-time statusline for Claude Code showing token usage, remaining budget, burn rate, and depletion time estimation. Helps users understand their usage patterns and predict when they might run out of their API budget. Features auto-updates daily.

## Key Features

- Token usage tracking
- Budget monitoring
- Burn rate calculation
- Depletion time estimation
- Auto-updates daily
