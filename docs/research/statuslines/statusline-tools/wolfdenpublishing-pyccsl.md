## Summary

Python Claude Code Status Line (PyCCSL, pronounced 'pixel'). Highly configurable with multiple display modes and themes including nord. Features powerline style support, performance metrics display, and configurable number formatting through environment variables.

## Key Features

- Multiple themes (nord, etc.)
- Powerline style support
- Performance metrics display
- Configurable number formatting
- Environment variable configuration

## Installation

```bash
python3 ~/.claude/pyccsl.py badge,model,perf-all-metrics,cost --theme nord
```
