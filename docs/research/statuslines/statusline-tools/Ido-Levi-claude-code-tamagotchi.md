## Summary

A digital friend that lives in your Claude Code statusline. Features a real-time behavioral enforcement system that monitors AI actions and detects violations. The virtual pet reacts to code activity and can interrupt operations when violations are detected.

## Key Features

- Virtual pet in statusline
- Behavioral enforcement system
- Real-time monitoring
- Operation interruption on violations
- Pet reacts to code activity
