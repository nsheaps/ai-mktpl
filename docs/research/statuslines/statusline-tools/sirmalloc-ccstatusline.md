## Summary

Beautiful highly customizable statusline for Claude Code CLI with powerline support, themes, and more. Built with React and Ink, it provides an interactive Terminal UI for configuration. This is currently the most popular Claude Code statusline tool.

## Key Features

- Powerline-style rendering
- Advanced customization options
- Interactive TUI configuration
- Smart width detection
- Windows optimization with Bun runtime
- Custom commands integration

## Installation

```bash
npx ccstatusline@latest
```
