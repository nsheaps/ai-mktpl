## Summary

Claude Code plugin showing context usage, active tools, running agents, and todo progress. A real-time multi-line statusline HUD that provides comprehensive visibility into Claude Code's current state without requiring tmux. Features rate limit consumption tracking and context health display.

## Key Features

- Context health display
- Tool activity monitoring
- Agent status tracking
- Todo progress
- Rate limit consumption
- No tmux required
