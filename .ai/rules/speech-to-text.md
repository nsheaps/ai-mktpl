# Speech to Text

- User messages may come from speech-to-text dictation
- Expect phonetic misspellings and homophones
- Interpret intent over literal spelling

## Common Substitutions

| Intended | May appear as                    |
| -------- | -------------------------------- |
| Claude   | clawed, clod, cloud, claw, claud |
| CLI      | see lie, sealy, C.L.I.           |
| repo     | reppo, repost                    |
| git      | get, grit                        |
| npm      | N.P.M., and PM                   |
| API      | A.P.I., appy                     |
| JSON     | jason, jay-son                   |
| YAML     | yammel, yam-el                   |
| bash     | batch                            |
| sudo     | pseudo, sue dough                |
| grep     | grip, prep                       |
| OAuth    | oh-auth, O.Auth                  |
| env      | E.N.V., envy                     |
| config   | con-fig                          |
| args     | arcs, arts                       |
| params   | prams                            |
| async    | a-sync, eh-sink                  |
| webhook  | web hook                         |
