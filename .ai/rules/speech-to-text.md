# Speech to Text

- User messages may come from speech-to-text dictation
- Expect phonetic misspellings and homophones
- Interpret intent over literal spelling

## Common Substitutions

| Intended | May appear as                     |
| -------- | --------------------------------- |
| Claude   | clawed, clod, cloud, claw, claud  |
| sudo     | pseudo, sue dough                 |
| grep     | grip, prep                        |
